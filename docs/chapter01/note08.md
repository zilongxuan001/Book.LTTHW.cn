# 如何让老师毫无保留教你？


很简单，你只需要让别人觉得——

## 1、教你是因为你的好奇与热情！

真正的好老师喜欢的学生是那种成绩不太好但真心好学的人，所谓真心好学是指对知识好奇与向往、遇见知识就喜悦、学习知识就沉浸，不拘束于课程大纲，期待认知边界的拓展。

但事实上，好老师不多，好学生也越来越少。「真正有悟性与灵性的学生越来越难遇到经师与人师的启发。另一方面，想要启迪学生心灵（而不只是传授知识）的经师与人师也越来越寂寞。」

我是很期待学生可以活出生命的价值与尊严——以他们真心所爱的方式，以他们深思后所选择的方式，而不是以他们的虚荣心或现实社会所鼓舞的那种方式；以他们全部的潜能，而不只是被一项技能（读书、写程式，或打乒乓球，etc）限制了其他生命潜能的发挥。

出自：[彭明辉：乐为人师（1）](https://www.douban.com/note/643667031/)和 [彭明辉：乐为人师（2）](https://www.douban.com/note/648977599/)

## 2、教你的感觉就是教对了人！

老师虽然知识丰富，但因「知识的诅咒」，他们往往不能理解新手的感受，忘却学生学习的难易，设计课程容易想当然，但如果此时有优秀的新手在旁，他就可以承担课程的设计师、体验师、运营者、顾问等角色，帮助老师设计出从易到难，符合新手认知的课程。

出自：[如何成为一名优秀新手？ - 读立写生](http://www.cnfeat.com/blog/2017/06/08/HowToBeGoodBeginner/)

## 3、教你的感觉很爽很舒服！

在一段时间后，我发现他身上有一种独特的气质，这种气质让人特别想要把自己的经验和思考分享给他，换句话说，就是特别想教他。那种感觉就像一记吸星大法打在身上，让人想要主动把自己这些年辛苦积累和总结下来的东西一股脑都传授给他。

出自：[求求你，让我教你](http://qiuyuexp.com/teach/)

## 4、教你很理所当然哇！

我曾遇到过很多企业家，他们中许多人也会做那些底层的工作，例如兼职，例如继续教别人打工，这是他们以前做的事情？但为什么他们还要做呢？我看到的是另一面：我看到是谦卑、我看到是好学。我看到的是他们正全身心地学习，正在观察并发现模式、正在磨练感觉、正在锐化头脑，正在等待恰当的时机。

出自：[《重来》作者：这样的人，才是我想帮的](http://t.cn/RXn8wtk)

## 5、教你就像在教自己！

1）人们愿意教「自己人」 ，你的品味、你的思想、你的兴趣，都可能让老师觉得你是「自己人」；2）人们愿意教助那些实际上是在教「自己人」，让自己的梦想，成为老师的梦想的一部分；3）人们愿意教有礼貌、积极向上、乐观与有奋斗精神的人，对美的欣赏与追求是人的天性或者本能；4）人们愿意教能够「教」的人，不要让「教」成为老师的负担。

出自：[「别人」凭什么帮你？ - 阳志平的网志](http://www.yangzhiping.com/psy/bieren.html)


## 小结

聪明人都明白这样一个真理；教助自己的惟一方法就是去教助别人。——埃伯德

祝你教的人越多，教你的人越多！
